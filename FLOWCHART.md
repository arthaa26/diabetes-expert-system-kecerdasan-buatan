## Instruksi Membuat Diagram Lebih Detail

Untuk membuat flowchart yang lebih profesional, Anda dapat menggunakan tools:

### 1. **Lucidchart** (Online, Web-Based)
- URL: https://www.lucidchart.com
- Kelebihan: Kolaboratif, Export ke berbagai format
- Buat diagram dari scratch atau import dari file

### 2. **Draw.io** (Online & Desktop)
- URL: https://draw.io
- Kelebihan: Gratis, offline support, banyak template
- Export: PNG, SVG, PDF, XML

### 3. **PlantUML** (Text-Based, Code)
- Gunakan syntax khusus untuk generate diagram
- Bisa diintegrasikan ke dalam repository

**Contoh PlantUML:**
```plantuml
@startuml
actor User
participant Browser
participant "Laravel Server" as Server
database Database

User -> Browser: Buka /konsultasi
Browser -> Server: GET /konsultasi
Server -> Database: SELECT gejalas
Database --> Server: Gejala List
Server --> Browser: Tampil Form + Popup
Browser -> User: Tampil Popup (Nama & Umur)
User -> Browser: Isi Data
Browser -> Server: POST /konsultasi/diagnose
Server -> Database: Forward Chaining + CREATE Activity
Database --> Server: Activity ID
Server --> Browser: Tampil Hasil
Browser -> User: Tampil Diagnosis

@enduml
```

### 4. **Mermaid** (GitHub Native)
Bisa langsung digunakan di markdown:

```mermaid
graph TD
    A[User Buka /konsultasi] --> B[Popup Nama & Umur]
    B --> C{Valid?}
    C -->|No| D[Show Error]
    D --> B
    C -->|Yes| E[Form Diagnosis]
    E --> F[Pilih Gejala]
    F --> G[Klik Diagnosa]
    G --> H[Validasi]
    H --> I[Forward Chaining]
    I --> J[Buat Activity]
    J --> K[Tampil Hasil]
    K --> L[User Lihat Diagnosis]
    L --> M[Admin Dashboard Update]
```

### 5. **Microsoft Visio** / **Draw.io Desktop**
- Untuk diagram offline dan file sharing
- Format: .vsdx, .xml, .drawio
